Redirect as a Service

Forward your domain as a Service.

  1. Point your (sub)domain which you want to redirect to go.redirectaas.ch using CNAME (or ALIAS).
  2. Create a TXT record below your (sub)domain as _redirect which contains a URL.

FAQ

Is there a rate limit?

Fair Use. As long as this does not explode, it should not matter.

CNAME does not work at root level and my dns provider does not support ALIAS.

Yes, then it does not work.
